# Token Harbor 機率與收藏長線玩法草案

> 狀態：產品規劃草案，只作後續方向記錄，未排期、未實作。
> 目標：加入抽卡感、運氣與概率變化，延長單人內容壽命，同時避免遊戲變成付費抽卡或鼓勵玩家浪費 Token。

## 1. 核心方向

Token Harbor 適合的不是傳統角色卡池，而是以下三個系統互相配合：

1. **命運航線**：每次出海從三張隨機航海情報中選一張，決定魚群、天氣、海盜、沉船與特殊事件的組合。
2. **海洋珍藏冊**：航海、研究和里程碑會送收藏包，抽到魚類、生態、港口、船員故事和傳說事件卡。
3. **漂流寶箱**：海上偶爾發現不同等級的沉船箱，回港後開出外觀、藍圖、材料或特殊訂單。

抽卡只佔長線內容的一部分。主要樂趣仍然是手動出海、追魚、處理威脅、配置船隻和建設漁港。

## 2. 設計底線

- 不設付費抽卡，不出售提高概率的道具。
- Sail Power 不設「直接十連抽」按鈕，必須透過真正航程產生發現與收藏。
- 船、設備、船員能力等核心戰力必須有確定取得路線，運氣只負責提早取得、外觀、故事或橫向玩法。
- 公開所有重要概率、保底進度和可能獎勵，不使用模糊的「大幅提升」。
- 隨機結果在出海時以本地種子確定，重新整理或強制關閉不能反覆重抽。
- 不設斷簽損失、限時卡池或錯過便永遠絕版的能力。
- 重複物品必須推進收藏，不讓玩家抽到完全無用的廢卡。
- 同時維持 Sail Power、Coins、Materials 三種主要資源上限，不新增鑽石、抽卡券、碎片粉塵等貨幣群。

## 3. 命運航線

### 基本流程

玩家按下出海後，先翻開三張航海情報，再選擇其中一條航線：

- **平潮航線**：普通魚較多、風險較低、收益穩定。
- **魚群航線**：魚量和稀有率提高，但保鮮箱更容易爆滿。
- **黑旗航線**：寶箱與稀有魚機會高，海盜也更強。
- **風暴航線**：高價魚和沉船較多，船體及返航風險提高。
- **迷霧航線**：不顯示完整事件，可能發現隱藏海域或一無所獲。
- **研究航線**：金幣較少，但容易遇到未完成的圖鑑觀察。

這不是單純擲骰。玩家會根據船體狀態、容量、大炮、訂單需求和目前收藏，自行選擇值得承受的風險。

### 情報準確度

- 初期只顯示「低、中、高」風險。
- 升級聲納會顯示稀有率範圍。
- 升級燈塔會揭示天氣與海盜類型。
- 船長專精可以重新整理其中一張情報，但每次出海最多一次。

如此一來，養成會改善玩家的判斷能力，不只是把數字變大。

## 4. 海洋珍藏冊

### 卡片類型

- **魚類卡**：物種、尺寸紀錄、特殊花紋和行為觀察。
- **生態卡**：潮汐、珊瑚、遷徙、深海發光與特殊天氣。
- **港口卡**：建築歷史、居民委託、地方傳說與港口外觀。
- **船員卡**：船員故事、專精回憶和人物關係。
- **遭遇卡**：海盜旗號、幽靈船、沉船、傳說魚蹤跡。

### 收藏包來源

- 完成首次航線、圖鑑階段、傳說線索或章節里程碑。
- 完成若干次有實際操作的手動航程。
- 完成研究、特殊訂單和遠征節點。
- 自動船只可帶回普通收藏包碎頁，不會直接抽到最高級收藏。

收藏包不是每日登入獎勵，也不能用 Sail Power 或真錢無限購買。

### 暫定開包規則

- 每包三張，至少一張為 Uncommon 或以上。
- 基礎概率暫定為 Common 55%、Uncommon 28%、Rare 12%、Epic 4%、Legendary 1%。
- 六包沒有 Rare 或以上，下一包保證至少一張 Rare。
- 三十包沒有 Legendary，下一包保證一張 Legendary。
- 數值只作方向參考，正式實作前需要以內容量和取得速度重新模擬。

### 重複卡處理

同一張卡分為「發現、精通、閃卡」三個收藏階段。重複卡會直接填入下一階段；全部完成後再次抽到，改為讓玩家從三張未完成的同級卡中選一張。整個過程不產生額外碎片貨幣。

完成一組收藏主要解鎖：

- 港口裝飾、船體塗裝、旗幟和水面特效。
- 魚類故事、船員對話與生態資料。
- 新航線情報或橫向能力，例如多看一張情報、標記一種魚。
- 小量且有上限的便利能力，不能成為唯一最強配置。

## 5. 漂流寶箱與打撈

寶箱在捕魚途中出現，玩家需要決定是否放棄追魚、承受海盜或容量風險去打撈。

### 寶箱層級

- **漂流木箱**：Coins、Materials、普通外觀部件。
- **沉船鐵箱**：設備藍圖、特殊訂單、稀有裝飾。
- **船長遺箱**：傳說線索、完整塗裝、永久收藏品。

寶箱不能直接售賣。開箱前顯示獎勵池；高階藍圖若長期抽不到，研究所會在固定進度後提供指定製作，確保核心功能不被運氣永久卡住。

## 6. 船員邂逅

不採用傳統船員角色卡池。玩家完成航程後，碼頭有機會出現訪港船員：

- 每次出現三名候選人，清楚顯示能力和性格。
- 玩家以 Coins 正常聘請，不需要抽中完整角色。
- 候選人的背景、專長組合與個人事件帶有隨機性。
- 連續若干次沒有適合目前船隊的職業，下一次必定出現所缺職業。
- 已有船員的重複邂逅會開啟故事或專精選項，不會變成角色碎片。

這樣保留「遇到好船員」的驚喜，同時不會讓玩家因抽不到核心角色而停滯。

## 7. 魚類運氣與傳說狩獵

### 個體變化

同一魚種可隨機出現尺寸、花紋、性格和特殊動作：

- 巨型、小型、傷痕、金色或夜光個體。
- 膽小魚更常逃離畫面；兇猛魚可能突然撞擊漁網。
- 特殊個體可放入水族館、研究或作為收藏紀錄，不只是售價更高。

### 稀有魚保護

- 每條航線顯示當前稀有度概率範圍。
- 連續五次手動航程沒有 Rare 或以上，之後每次逐步增加小量概率，捕到後重置。
- Legendary 不完全依賴直接抽中。失敗遭遇會留下線索，收集滿十條線索便開啟一次確定的傳說狩獵。
- 自動船不會消耗或推進最高級保底，避免掛機取代手動玩法。

## 8. 風險合約

出海前可以選擇一份不收額外貨幣的合約：

- **安全返航**：收益較低，船體不會因單一事件直接全毀。
- **滿載而歸**：容量提高，但返航前不能中途卸貨。
- **賞金航程**：海盜必定出現，擊退後有較高寶箱概率。
- **風暴追蹤**：颶風風險明示，成功返航會獲得天氣收藏和高價魚群。

原有 5% 颶風可以保留，但長線版本應透過航海情報顯示風險，並讓燈塔、船體、保險或路線選擇提供對策。隨機災難要產生故事與判斷，不只是無預警清空漁獲。

## 9. 輪替海況與非限時季節

### 世界海況

每週以本地種子產生一組全球海況，例如：

- 月光鮪魚遷徙。
- 海盜勢力轉移。
- 珊瑚產卵潮。
- 深海暖流或長霧期。

錯過一週不會損失獎勵。海況只令舊航線產生不同玩法和收集目標。

### 航海紀事

每六至八週加入一個小型主題紀事，包含一組事件、收藏頁和傳說線索。紀事結束後會進入港口檔案館，回歸玩家仍可慢慢完成，不使用永久絕版製造焦慮。

## 10. 長線節奏

| 時間層 | 隨機變化 | 長期成果 |
| --- | --- | --- |
| 每次出海 | 三選一命運航線、魚群、威脅、寶箱 | 漁獲、收藏進度、故事事件 |
| 每數次航程 | 收藏包、船員邂逅、特殊訂單 | 卡冊套組、專精、外觀 |
| 每週 | 世界海況與遷徙 | 重玩舊航線、補齊圖鑑 |
| 每 6–8 週 | 航海紀事 | 永久故事章與收藏頁 |
| 每季度或大型更新 | 新群島章節 | 新規則、船型、傳說狩獵 |

## 11. 建議優先順序

### P0：先改善現有概率體驗

1. 顯示航線稀有率、海盜和天氣風險。
2. 為稀有魚加入軟保底，傳說魚改用線索保底。
3. 隨機結果使用可重現種子，防止重新整理洗結果。
4. 確保每次負面隨機事件都有情報或養成對策。

### P1：建立長線主體

1. 命運航線三選一。
2. 海洋珍藏冊與無廢卡重複規則。
3. 魚類尺寸、花紋與性格變化。
4. 漂流寶箱與指定製作保底。

### P2：擴充營運內容

1. 船員邂逅與故事專精。
2. 每週世界海況。
3. 可永久回玩的航海紀事。
4. 分支遠征內的事件牌庫。

## 12. 商業化界線

- 可以直接售賣清楚可見的船體塗裝、港口主題、音樂或裝飾包。
- 可以售賣永久內容章節，但不能把已擁有內容拆成隨機箱再售賣。
- 不出售收藏包、保底進度、稀有率加成、Legendary 線索或戰力卡。
- 不把 Codex Token 消耗包裝成賭注；Token 只轉為 Sail Power，Sail Power 用於玩家本來就會進行的航程。

## 13. 成功標準

- 玩家第 30 日仍會因航線情報而作出不同選擇。
- 玩家第 60 日回到舊港口，仍有新個體、世界海況和收藏目標。
- 玩家第 90 日有一至兩段可以講述的幸運航海故事，而不只是更大的數字。
- 大部分玩家能預測何時取得下一個高稀有獎勵，不會覺得系統暗中操控結果。
- 沒有玩家因抽不到核心船、設備或船員而無法推進章節。
- 即使完全不付費，所有概率玩法和永久收藏仍可正常完成。

## 14. 目前建議結論

最適合 Token Harbor 的組合是「命運航線 + 海洋珍藏冊 + 漂流寶箱」，船員只做訪港邂逅，不做角色卡池。

這個方向同時提供短局驚喜、中期收集和長期內容輪替。真正戰力保持確定成長，抽卡感主要服務收藏、故事與橫向選擇，避免遊戲變成另一款換皮抽卡手游。
